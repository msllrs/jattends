# Jattends Changelog

## [Initial Version] - 2026-02-28

- List all active Claude Code sessions grouped by status (Waiting / Working / Ready)
- Fuzzy search by project name, directory path, or session ID
- Jump to the correct terminal window on Enter
- Works with Ghostty, iTerm2, Terminal.app, kitty, Alacritty, WezTerm, and more
- Automatic cleanup of stale and dead sessions
